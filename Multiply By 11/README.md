## Multiply By 11
> **Today Kevin’s Teacher taught Kevin, Multiplication table of 11.After reaching home, Kevin thought of multiplying the numbers with 11 and
finding the answers.Help Kevin finding the answers as soon as possible.**

**Input Format**<br/>
First line of input contains T number i.e number of test cases, each test case followed by T numbers.<br/>

**Output Format** <br/>
For each test case, Print a single integer.<br/>

**Constraints**<br/>
0<=T<=100<br/>
0< Number of Digits in Integer <=1000000<br/>

**Sample Input**<br/>
3<br/>
8<br/>
12<br/>
15<br/>

**Sample Output**<br/>
88<br/>
132<br/>
165<br/>
