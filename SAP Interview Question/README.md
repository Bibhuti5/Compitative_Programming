## Shift 0s to end

> **You are given an array. Your task is to shift all the 0s to the end.

**Input Format**<br/>
Only one line of input containing numbers of array<br/>

**Output Format** <br/>
Print the shifted array.

**Constraints** <br/>
Time complexity should be O(n)
Do not use any other array.

**Sample Input**<br/>
1 3 2 0 5 0 6 0<br>

**Sample Output**<br/>
1 3 2 5 6 0 0 0
